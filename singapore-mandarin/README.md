# Singapore Mandarin — 新加坡華語

將文字轉為新加坡華語（Huayu）——新加坡華人社群的官方與日常書面語變體，受閩南語、馬來語、英語長期接觸形塑。涵蓋 SG 行政生活詞彙（組屋/德士/樂齡/客工/新生水/易通卡）、閩南語借詞（幾時/不懂/做工/青菜/頭家/燒）、馬來語借詞（甘榜/巴剎/沙爹/atas/bodoh）、詞序倒置（先後置：吃先）、特殊量詞（一粒/一間）、與數字單位差異（99 千 = 99,000）。

## 功能

依照 checklist 將文字轉為 SG 華語風格，涵蓋以下模式：

| 類別 | 項目 |
|------|------|
| **語法模式** (G1-G6) | 先後置、字=5分鐘、拜=星期、千作為單位、動詞重疊+一下、有+動詞 |
| **語尾助詞** (P1-P5) | 啦、咯、嘛/而已、咧/leh、Singlish 跨界（lah/lor/mah） |
| **詞彙模式** (V1-V7) | SG 行政生活詞、閩南語借詞、馬來語借詞、英語音譯與字母詞、粵語借詞、咖啡店隱語、文雅偏好 |
| **結構模式** (S1-S5) | 中英 code-mix、中閩短句插入、Singlish 粒子、字母詞保留、千-單位數字 |

支援三段強度：

| 強度 | 說明 |
|------|------|
| **輕** | SG 行政詞（組屋/德士/樂齡）+ 字母詞（MRT/HDB）+ 英語音譯 + 少量 code-mix + 千-單位 + 文雅偏好。無閩南語借詞、無跨界粒子。像《聯合早報》報導或政府文告。 |
| **中**（預設） | 輕度 + 全套語法 + 閩南語借詞核心（幾時/不懂/做工/怕輸/頭家）+ 馬來語借詞 + 中度 code-mixing。像部落格、紅螞蟻式半正式文字。 |
| **重** | 中度 + 咖啡店隱語（踢球/釣魚/水草）+ Singlish 跨界粒子（lah/lor/leh）+ 密集 code-mix + 閩南語短句插入。社交媒體、WhatsApp 聊天風格。仍保持華語骨架，不倒向 Singlish。|

## 使用方式

```
/singapore-mandarin path/to/article.md
```

**觸發詞：** `/singapore-mandarin`、「寫成新加坡華語」、「新加坡華語化」、「翻成新加坡華語」、「SG 華語」、「Huayu」

支援任何語言輸入，轉為 SG 華語輸出。

## 運作流程

1. **載入** — 讀取檔案或行內文字
2. **轉換** — 依 checklist 逐項套用 SG 華語模式
3. **輸出** — 另存為 `_singapore` 後綴新檔，附 3-5 組前後對比與借詞 glossary

## 範例

| 原文（標準中文） | SG 華語（Medium） |
|------|--------|
| 我坐出租車去市場。 | 我坐德士去巴剎。 |
| 他上班去了。 | 他做工去了。 |
| 什麼時候回家？ | 幾時回家？ |
| 我不知道。 | 我不懂。 |
| 讓我想一下。 | 讓我想想一下。 |
| 你先吃。 | 你吃先。 |
| 這個很時髦。 | 這個很 atas。 |
| 今年的年終獎金很高。 | 今年花紅 very 高。 |

## 設計原則

- **華語基底，非 Singlish**：SG 華語與 Singlish 是雙胞胎——前者華語底、後者英語底。本 skill 嚴格保持華語語法骨架（我/你/他、是/不/的/了），code-mixing 只替換內容詞。
- **文雅偏好**：SG 官方華語系統性地選擇更人文的詞（樂齡/客工/建國一代），這是真實的語用特徵而非單純翻譯偏好。
- **千-單位**：10,000 到 999,999 用千，不用萬，這是 SG 華語的**結構性**特徵而非選擇。
- **簡繁體跟隨輸入**：SG 官方用簡體，但 skill 預設不強制轉換，尊重使用者輸入；如需強制簡體可另外指定。
- **排除 caricature**：以聯合早報、紅螞蟻、官方文告為典範，不做誇張「SG 腔」戲仿。
