# Wenyan-Xianqin — 改寫為先秦諸子體

將現代中文改寫為先秦諸子散文風格，追求「辭約義豐、一字千金」的極致簡鍊。

## 功能

模擬《論語》《孟子》《莊子》《韓非子》《左傳》的語感。這是四種文言風格中壓縮最極端的——目標產出字數為原文的 **30-50%**。

20 項 checklist 涵蓋：

- **語法 (G1-G6)** — 也字判斷（無繫詞「是」）、極致省略主語、主題述語、假設因果、對偶壓縮、反問定式（不亦…乎）
- **詞彙 (V1-V7)** — 單音節詞、語氣詞（矣/乎/哉/也/焉）、代詞、虛詞、否定詞（弗/未/毋/莫）、去連詞、去把被句
- **風格 (S1-S4)** — 格言凝鍊（4-8 字為一句）、寓言取譬、對話標記（X曰）、微言大義
- **時代標記 (R1-R3)** — 禁多音節連詞、古典時地詞、四言為基

> **注意：** 多數實用場景（論述、提案、正式書寫）建議使用 [wenyan-guwen](../wenyan-guwen/)（唐宋古文體）。本技能適合刻意追求先秦語感的場合。

## 使用方式

```
/wenyan-xianqin path/to/article.md
```

**觸發詞：** `/wenyan-xianqin`、「先秦體」、「諸子體」、「論語體」、「莊子體」

## 運作流程

1. **載入** — 讀取檔案或行內文字
2. **評估** — 識別核心思想、適合的子形式（語錄/論辯/寓言/記事）
3. **改寫** — 依 checklist 逐項轉換，遵循 6 條改寫原則，目標壓縮至 30-50%
4. **審查** — 二輪自檢：重掃 checklist + 字數比檢查 + 通讀語感
5. **輸出** — 另存為 `_xianqin` 後綴新檔
