# Wenyan-Guwen — 示範

展示 `wenyan-guwen` skill 將現代中文改寫為唐宋古文體的效果。源文來自 [`examples/source-texts/`](../../examples/source-texts/)。

本 skill 為單一風格輸出（無強度分層），仿韓愈、柳宗元、歐陽修、蘇軾的議論散文語感。

---

## 範例：差不多先生傳（短摘錄）

源文：[mr-chabuduo.md](../../examples/source-texts/mr-chabuduo.md)（胡適 1924，公有領域）

### 原文

> 你知道中國最有名的人是誰？提起此人，人人皆曉，處處聞名。他姓差，名不多，是各省各縣各村人氏。你一定見過他，一定聽過別人談起他。差不多先生的名字天天掛在大家的口頭，因為他是中國全國人的代表。
>
> 差不多先生的相貌和你和我都差不多。他有一雙眼睛，但看的不很清楚；他有兩隻耳朵，但聽的不很分明；有鼻子和嘴，但他對於氣味和口味都不很講究。他的腦子也不小，但他的記性卻不很精明，他的思想也不很細密。
>
> 他常常說：「凡事只要差不多，就好了。何必太精明呢？」
>
> 他小的時候，他媽叫他去買紅糖，他買了白糖回來。他媽罵他，他搖搖頭說：「紅糖白糖，不是差不多嗎？」
>
> 他在學堂的時候，先生問他：「直隸省的西邊是哪一省？」他說是陝西。先生說：「錯了。是山西，不是陝西。」他說：「陝西同山西，不是差不多嗎？」

### 改寫（唐宋古文體）

> 差不多者，不知何郡人也。亦不詳其字。其姓氏行第，皆莫能明。然天下之人，無不知其名者。蓋舉國之民，皆其類也。
>
> 其為人也，目不甚明，耳不甚聰，鼻不辨臭，口不知味。腦有所具，而記性頗鈍，思慮亦疏。
>
> 常語人曰：「凡事差不多即可，何必過為精明邪？」
>
> 少時，母使買紅糖，乃持白糖歸。母責之，搖首曰：「紅糖白糖，非差不多乎？」
>
> 及就學，師問：「直隸之西，何省也？」對曰：「陝西。」師曰：「非也，是山西，非陝西。」不多曰：「陝西與山西，非差不多乎？」

**字數比**：原文 ~330 字 → 改寫 ~200 字（≈61%）

## 改寫重點

| 規則 | 出處 | 改寫 |
|---|---|---|
| G1 主題—評論句式 | 「他的相貌和你和我都差不多」 | 「其為人也」+ 後接評論 |
| G3 讓步轉折 | 「但他對於氣味和口味都不很講究」 | 「鼻不辨臭，口不知味」（駢句轉折） |
| G4 長短交錯 | 整段平鋪 | 「腦有所具，而記性頗鈍，思慮亦疏」（4+4+4） |
| G6 駢散結合 | 「眼睛、耳朵、鼻子、嘴」 | 「目不甚明，耳不甚聰，鼻不辨臭，口不知味」（駢） |
| G7 判斷句 | 「他是…代表」 | 「皆其類也」 |
| V1 語氣詞 | 「呢？」 | 「邪？」（更具古文味的反問） |
| V3 單音節動詞 | 「看」「聽」 | 「明」「聰」 |

## 風格特色

- **駢散結合**：四字句與長句交錯，避免完全駢偶（不像賦體）
- **議論性開場**：可加「夫」「蓋」起興
- **長短句交錯**：避免一律 4 字或一律長句
- **轉折詞**：「然」「而」「則」「若夫」（不用「但是」「不過」）
- **「也」字判斷**：「X 者，Y 也」「夫 X 者」
- **論贊段落**（可選）：「論曰」「贊曰」「太史公曰」

> **使用建議**：在 4 個 wenyan skill 中，guwen 最適合**論述、提案、正式書寫**等實用場合，文白比例最易為現代讀者接受。
