# Wenyan-Guwen — 改寫為唐宋古文體

將現代中文改寫為唐宋古文運動風格的文言散文。

## 功能

模擬韓愈、柳宗元、歐陽修、蘇軾的行文風格，追求「文以載道、氣韻生動」的質感。22 項 checklist 涵蓋：

- **語法 (G1-G7)** — 主題句式、假設因果、讓步轉折、長短交錯、設問反問、駢散結合、也字判斷
- **詞彙 (V1-V7)** — 語氣詞、虛詞、動詞精簡、代詞、去藻飾、否定詞、去連詞贅語
- **風格 (S1-S5)** — 文以載道開篇、用典、氣韻生動、情理交融、收束有力
- **時代標記 (R1-R3)** — 消除把/被、古典時地詞、古典句末模式

## 使用方式

```
/wenyan-guwen path/to/article.md
```

**觸發詞：** `/wenyan-guwen`、「寫成古文」、「寫成文言文」、「唐宋風格」、「韓愈體」、「蘇軾體」

## 運作流程

1. **載入** — 讀取檔案或行內文字
2. **評估** — 識別核心論點、現代用法、適合的文體（論/序/記）
3. **改寫** — 依 checklist 逐項轉換，遵循 6 條改寫原則
4. **審查** — 二輪自檢：重掃 checklist + 通讀語感
5. **輸出** — 另存為 `_guwen` 後綴新檔
