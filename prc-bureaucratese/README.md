# PRC Bureaucratese — 中共黨政公文體

將文字轉為中共黨政公文體（PRC party-state bureaucratese）——中國大陸黨政機關文件、新華社通稿、領導人講話、《人民日報》社論的制度化書面語風格。融合高頻動詞（落實/加強/推進/深化）、副詞堆疊（紮實推進/深入貫徹/全面落實）、四字格對偶（不忘初心、牢記使命）、數字化政治套語（三個務必/四個全面/五位一體）、新時代意識形態詞彙（新時代/新發展格局/中國式現代化/偉大復興）、開頭結尾套話模板。

## 功能

依照 checklist 將文字轉為中共公文體，涵蓋以下模式：

| 類別 | 項目 |
|------|------|
| **詞彙模式** (V1-V6) | 高頻動詞、高頻副詞、核心名詞概念、新時代政治詞彙、數字化政治套語、核心意識形態序列 |
| **結構模式** (S1-S6) | 三件套排比、四字格對偶、既要又要雙重目的、動詞堆疊、名詞化升級（XX性/化）、修飾詞層層堆疊 |
| **套話模板** (T1-T4) | 開頭套話、結尾套話、連接性套話、強調性套話 |

支援三段強度：

| 強度 | 說明 |
|------|------|
| **輕** | 基本動詞（落實/加強/推進）+ 副詞堆疊（紮實/深入/切實）+ 簡單動賓搭配。無政治意識形態、無數字化套語。像地方政府日常通知、部門內部紀要。 |
| **中**（預設） | 輕度 + 新時代詞彙 + 排比/四字格/動詞堆疊 + 開頭結尾模板。像部委會議紀要、省級工作報告。 |
| **重** | 中度 + 完整意識形態序列（馬列毛鄧三科習）+ 數字化套語全開（四個全面/五位一體/兩個維護）+ 多層修飾詞堆疊。經典黨八股，像黨報社論、政府工作報告。|

## 使用方式

```
/prc-bureaucratese path/to/article.md
```

**觸發詞：** `/prc-bureaucratese`、「寫成公文體」、「改成中共公文體」、「黨政公文化」、「黨八股化」、「官樣文章」、「PRC official style」

## 運作流程

1. **載入** — 讀取檔案或行內文字
2. **轉換** — 依 checklist 套用公文體模式
3. **輸出** — 另存為 `_bureaucratese` 後綴新檔，附 3-5 組前後對比與套語 glossary

## 範例

| 原文 | 公文體（Medium） |
|------|--------|
| 我們要更努力工作。 | 要進一步紮實推進各項工作落實到位。 |
| 這個政策很重要。 | 這一政策具有全局性、戰略性、根本性的重大意義。 |
| 請大家配合。 | 希各單位切實加強組織領導，壓實責任、強化措施。 |
| 我們做得不夠好。 | 工作中仍存在一定差距與不足，需進一步深化。 |
| 要重視環境問題。 | 要堅持綠水青山就是金山銀山的理念，深入推進生態文明建設。 |
| 謝謝大家。 | 以優異成績為實現中華民族偉大復興貢獻力量。 |

> 完整示範（3 強度 × 2 源文）請見 [examples/demo.md](./examples/demo.md)。

## 設計原則

- **風格模仿，非政治立場**：和 repo 其他 skill 一樣（如 wenyan-xianqin 模仿先秦諸子並非提倡諸子思想），本 skill 忠實複製書面風格，不代表使用者立場。
- **形式性即特徵**：黨政公文體的**冗餘、堆疊、套話**不是缺陷而是風格特徵；skill 不去「優化」簡化。
- **數字化套語不改寫**：四個全面、五位一體、兩個維護等是**定型短語**，只複製不改寫。
- **毛澤東《反對黨八股》自嘲性**：毛 1942 年批判的 8 罪狀正是當代公文的典型特徵；這種反諷是歷史肌理的一部分。
- **新時代為預設時代**：聚焦 2010s-2020s 習時代為主，Light 涵蓋更廣泛通用風格。
- **與 roc-bureaucratese-classic 明確區分**：ROC 公文以文言色彩濃厚（著即/仰即/爰/茲/台端），本 skill 以馬列式政治話語為主，共享「中文公文」大類但內涵完全不同。
