# Westernise — 為中文添加翻譯腔

將自然的繁體中文改寫為帶有「翻譯腔」風格的文字，模擬從西方語言過度直譯的效果。

## 功能

依照 25 項 checklist 將中文改為歐化風格：被動句、抽象主語、從屬子句後置、「的」字堆疊、冗贅介詞、萬能動詞、代名詞補回、時態標記等。適用於風格實驗、創意寫作，或展示歐化模式的教學用途。

支援三段改寫強度：

| 強度 | 說明 |
|------|------|
| **輕** | 僅套用 G1-G5 語法模式，翻譯腔若隱若現 |
| **中**（預設） | 全套 G + V 模式，明顯翻譯腔 |
| **重** | 全套模式含 S 類結構，讀起來像機器直譯 |

## 使用方式

```
/westernise path/to/article.md
```

**觸發詞：** `/westernise`、「加翻譯腔」、「加歐化」、「寫成翻譯腔」、「讓它聽起來像翻譯」

## 運作流程

1. **載入** — 讀取檔案或行內文字
2. **改寫** — 依 checklist 逐項將自然中文改為歐化風格
3. **輸出** — 另存為 `_westernised` 後綴新檔，附 3-5 組前後對比
