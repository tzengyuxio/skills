# Dewesternise — 示範

展示 `dewesternise` skill 如何辨識並改寫歐化中文（翻譯腔）。本 skill 沒有強度分層，採「Before / After」對照。

> **源文說明**：本 demo 的 Before 段落為示範用的歐化中文範本（自編），不放入共用 [`source-texts/`](../../examples/source-texts/) — 因為「有問題的文本」不適合作為他用。

---

## 範例一：科技評論段落（多處歐化）

### Before（歐化中文）

> 在現代社會中，科技的快速發展對於人們的日常生活產生了深遠的影響。隨著智慧型手機的普及化，越來越多的人們花費了大量的時間在社交媒體的使用上。一項由哈佛大學所進行的研究指出，過度的螢幕時間可能會導致睡眠品質的下降。專家們建議，使用者應該對自己的螢幕使用時間進行嚴格的管理，以避免被各種通知所打擾。

**問題標記：**

| 編號 | 模式 | 出處 |
|---|---|---|
| V1 | 介詞堆疊 | 「對於人們的日常生活」 |
| G3 | 抽象名詞作主語 | 「科技的快速發展…產生影響」 |
| V8 | 萬能動詞 | 「進行嚴格的管理」「進行了研究」 |
| G5 | 們字濫用 | 「人們」「專家們」 |
| G1 | 被動句 | 「被各種通知所打擾」 |
| V5 | -化／-性後綴 | 「普及化」「下降」 |
| V4 | 「一個」冗贅冠詞 | 「一項由…所進行的研究」 |
| V2 | 「作為／透過」浮濫 | （改寫版會避免重蹈） |

### After（自然中文）

> 智慧型手機普及之後，大家花在社群媒體上的時間越來越多。哈佛大學一項研究發現，螢幕時間過長會讓睡眠品質變差。專家建議用戶自己管好用螢幕的時間，免得通知一直打斷你。

**改寫重點：**

- 「在現代社會中」純屬填充，刪掉
- 「對於…產生了深遠的影響」改成主動句，主語回到人
- 「進行嚴格的管理」→「管好」，恢復單一動詞
- 「們」全刪，靠語境表達複數
- 「被…所打擾」改主動：「打斷你」

## 範例二：產品介紹（「的」字鏈與長前飾語）

### Before

> 這是一個由台灣團隊所開發的、強調隱私保護的、跨平台的密碼管理工具。它能夠對使用者所輸入的密碼進行加密，並透過雲端進行同步。

### After

> 這是台灣團隊做的密碼管理器，主打隱私保護，支援跨平台。輸入的密碼會加密，再用雲端同步。

**改寫重點：**

- 三層「的」字鏈拆成短句
- 「對…進行加密」→「加密」
- 「透過雲端進行同步」→「用雲端同步」
- 「能夠」可省（中文常省可能性助動詞）

## 範例三：書評段落（從屬子句倒置）

### Before

> 雖然這本書因為它所探討的議題的廣泛性而被許多評論家所推崇，但是它在敘事節奏的處理上仍然存在著一些值得商榷的問題。

### After

> 這本書議題涵蓋很廣，所以不少評論家給了好評；不過敘事節奏的處理還是有些可議的地方。

**改寫重點：**

- 「雖然…但是…」改成兩個短句以分號連接
- 「因為…而被…所…」三層歐化拆解
- 「存在著一些…問題」→「有…的地方」（恢復中文存在句式）
- 「值得商榷的問題」是名詞化＋萬能動詞，改成形容詞性短語

## 改寫原則速查

| 原則 | 怎麼用 |
|---|---|
| 把長句切成短句 | 中文一句 12-20 字最舒服，超過要動刀 |
| 主語回到人 | 抽象名詞（「研究」「政策」）作主語多半要改 |
| 萬能動詞還原 | 「進行討論」→「討論」；「作出決定」→「決定」 |
| 們字省略 | 「人們」「同學們」絕大多數可刪 |
| 「的」字限縮 | 「強調 X 的 Y 的 Z」拆成 2-3 個短句 |
| 被動改主動 | 中文偏好主動，「被…所…」尤其要改 |
| 「說人話」測試 | 唸出來覺得彆扭就是歐化 |
