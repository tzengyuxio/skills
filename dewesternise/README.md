# Dewesternise — 去除歐化中文痕跡

審閱並改寫繁體中文，移除因直譯西方語言句法而產生的「翻譯腔」。

## 什麼是歐化中文？

歐化中文（西化中文、翻譯腔）指直譯歐美語言句法而未考慮中文本身表達習慣所形成的不自然文字。常見於翻譯稿、長期使用外語者的中文書寫，以及 AI 生成的中文。

余光中在〈怎樣改進英式中文？——論中文的常態與變態〉一文中，最早系統性地指出這個問題。他觀察到現代中文受英文影響，在句法、用詞、語序上漸漸偏離中文的自然表達，形成一種「英式中文」。

### 常見歐化模式

歐化中文的問題可分為三大類：

**語法層面：** 被動句濫用（「被認為是」取代「大家認為」）、抽象名詞作主語（「收入的減少導致…」取代「因收入減少而…」）、從屬子句倒置（先主句後原因，違反中文先因後果的語序）、「們」字濫用（「政府們」「專家們」）、代名詞過度補回（中文習慣省略可推斷的主語）。

**詞彙層面：** 介詞堆疊（「關於」「對於」「就…而言」）、「作為」「透過」的浮濫使用、「的」字連鎖修飾語、「一個」充當冗贅冠詞、「-性」後綴（「可行性」「合理性」）、副詞加「地」（「慢慢地」）、萬能動詞（「進行討論」取代「討論」、「作出決定」取代「決定」）、「最…之一」的迂迴表達、多餘的時態標記（「曾」「已」「了」「過」堆疊）。

**結構層面：** 語意重疊（「提前預告」「互相合作」）、超長前飾修飾語（將多句壓成名詞前的定語鏈）。

### 「說人話」原則

判斷是否歐化的最簡單方法：把句子用嘴巴說一次。如果說出來覺得彆扭，多半就是歐化了。自然的中文是先口語化表達，再轉為書面語。去歐化不等於去書面語——正式文體可以保留，但句法應該跟著中文的思路走。

## 功能

掃描文字中的歐化寫法，然後改寫為自然的中文表達。

25 項 checklist，分為 Grammar (G1-G8)、Vocabulary (V1-V14)、Structure (S1-S3) 三大類，每項標記 [Hard]（必改）或 [Soft]（視語境），附 Before/After 範例。

## 使用方式

```
/dewesternise path/to/article.md
```

**觸發詞：** `/dewesternise`、「去歐化」、「太歐化了」、「翻譯腔」、「去翻譯腔」、「改掉西化」

## 運作流程

1. **載入** — 讀取檔案或行內文字
2. **掃描** — 逐項比對 25 項 checklist，標記歐化段落
3. **報告** — 以表格呈現問題位置、類型、說明
4. **改寫** — 依 11 條改寫原則重寫標記段落
5. **審查** — 二輪自檢：重掃 checklist + 通讀語感確認
6. **輸出** — 另存為 `_dewesternised` 後綴新檔
