# Taiwan Mandarin — 台灣國語

將文字轉換為台灣國語——一種由國語長期接觸台灣閩南語而產生的社會方言，融合閩南語借詞、日語借詞、擬音字（偶/粉/素/降）、注音文（ㄉ/ㄇ/ㄅ/3Q），以及豐富的語尾助詞（啦/喔/齁/耶）。

## 功能

依照 checklist 將文字轉為台灣國語風格，涵蓋以下模式：

| 類別 | 項目 |
|------|------|
| **語法模式** (G1-G8) | 有+動詞完成體、給我句式、會+形容詞、強化詞堆疊、動詞重疊、是說/是怎樣、讓步句、主題句式 |
| **語尾助詞** (P1-P8) | 啦（緩和）、喔（提醒）、耶/欸（驚喜）、齁（求同）、啊（開場）、捏（撒嬌）、咧（比較）、哦（接話） |
| **詞彙模式** (V1-V7) | 台語借詞核心集與擴展集、日語借詞、擬音字、注音文、網路次文化用語、口語詞彙置換 |
| **結構模式** (S1-S4) | 顏文字、語氣拉長、感嘆詞、標點擴增 |

支援三段強度：

| 強度 | 說明 |
|------|------|
| **輕** | 核心語法＋基本語尾助詞＋少量核心台語借詞。無擬音字、無注音文。像一般台灣人日常 LINE 聊天。 |
| **中**（預設） | 全套語法 ＋ 全套助詞 ＋ 台語/日語借詞 ＋ 擬音字偶用（粉/素/醬）＋ 顏文字。像 PTT 八卦板輕鬆貼文。 |
| **重** | 擬音字全開（偶/偶棉/降/釀）＋ 注音文（ㄉ/ㄇ/ㄅ/3Q）＋ 密集網路次文化用語。2000 年代無名小站、奇摩即時通風格。 |

## 使用方式

```
/taiwan-mandarin path/to/article.md
```

**觸發詞：** `/taiwan-mandarin`、「寫成台灣國語」、「台灣國語化」、「翻成台灣國語」、「台灣腔」、「台式中文」

支援任何語言輸入，轉為台灣國語輸出。

## 運作流程

1. **載入** — 讀取檔案或行內文字
2. **轉換** — 依 checklist 逐項套用台灣國語模式
3. **輸出** — 另存為 `_taiwan` 後綴新檔，附 3-5 組前後對比與借詞表

## 範例

| 原文 | 台灣國語（Medium） |
|------|--------|
| 這家店的食物非常好吃。 | 這家店超好吃的啦！ |
| 你去過日本嗎？ | 你有去過日本齁？ |
| 今天很冷。 | 今天粉冷耶，會冷死。 |
| 我不知道是不是真的。 | 偶ㄅ知素ㄅ素真ㄉ（Heavy） |
| 他真的很討厭。 | 他真的有夠機車！ |
| 不好意思，借過。 | 歹勢歹勢，借過一下。 |

> 完整示範（3 強度 × 2 源文）請見 [examples/demo.md](./examples/demo.md)。

## 設計原則

- **真實語言變體，非醜化**：台灣國語是台灣社會真實存在的口語變體，skill 以語言學尊重為前提設計
- **排除 caricature**：不做鄉土劇誇張化、不做戲謔模仿、不 stereotype 閩南語使用者
- **強度是連續光譜**：從輕度（淺口語）到重度（BBS 鄉民文化），使用者自選
- **書面 ≠ 口語**：skill 轉換的是**書面呈現**——擬音字與注音文都是把口語特徵寫下來的慣例，不是標準正字
