# ROC Bureaucratese (Classic Era) — 中華民國 1960s-80s 公文體

將文字轉為 1960-80 年代中華民國公文體——白話化改革前的半文言制度化書面語。融合文言單字（之/於/以/係/為/俾/爰）、文言連接詞（茲/惟/嗣/俟/頃/殆/甫）、階層敬語系統（鈞/貴/本/職/台端/鈞長）、固定程序套語（查照/鑒核/核備/照辦/奉悉/敬悉/業經/迭經）、傳統結尾慣例（此致 XX 鈞鑒、為荷、特此函達）。

## 功能

依照 checklist 將文字轉為民國舊式公文體，涵蓋以下模式：

| 類別 | 項目 |
|------|------|
| **詞彙模式** (V1-V9) | 文言單字替換、連接詞/語氣詞、稱謂系統、引述語、經辦語、准駁語、期望語、文言形容詞副詞、特有成語套詞 |
| **結構模式** (S1-S4) | 主旨-說明-辦法格式、電報體式省字、文言疑問/反詰句、四字格公文習語 |
| **套話模板** (T1-T4) | 開頭套話、結尾套話、特定文類套語、敬語修飾 |

支援三段強度：

| 強度 | 說明 |
|------|------|
| **輕** | 基本文言單字替換（之/於/以/係）+ 稱謂系統 + 基本期望語+ 簡單開頭結尾。類似 1980s 後期向白話化過渡的風格，可讀性高。 |
| **中**（預設） | 輕度 + 完整文言連接詞（惟/爰/茲/嗣/俟）+ 引述語/經辦語/准駁語/期望語 + 文言形容詞副詞 + 主旨-說明-辦法格式 + 四字格習語。典型 1970-80 年代中期 ROC 公文。|
| **重** | 中度 + 電報體式省字 + 文言疑問/反詰句 + 全段去白話化 + 擴充敬語全用。1960 年代至 1970 初期的重度文言公文，接近電報體。|

## 使用方式

```
/roc-bureaucratese-classic path/to/article.md
```

**觸發詞：** `/roc-bureaucratese-classic`、「寫成民國公文體」、「改成中華民國公文」、「舊式公文化」、「民國公文化」、「仰即/著即 公文體」、「ROC classic bureaucratese」

## 運作流程

1. **載入** — 讀取檔案或行內文字
2. **轉換** — 依 checklist 逐項套用舊式公文體模式
3. **輸出** — 另存為 `_roc_classic` 後綴新檔，附 3-5 組前後對比與套語 glossary

## 設計原則

- **風格模仿，非政治立場**：和 repo 其他 skill 一樣，本 skill 忠實複製風格，不涉政治評價。
- **文言濃度即特徵**：不去「白話化優化」——大量單字文言、省字電報體、敬語層級密佈**正是**此體的風格核心。
- **敬語系統嚴格層級**：鈞（上級機關）/ 貴（平行或下級）/ 本/職（自稱）/ 台端（對人民）有明確分工，誤用是非母語感的明顯跡象。
- **套語程序化**：查照、鑒核、核備、奉悉、敬悉、已悉 有嚴格的階層適用規則，不得混用。
- **目標時代 1960s-80s**：1993 年公文格式改革後 ROC 公文逐步白話化，現代 ROC 公文不在本 skill 範圍（日後若有需求可做另一個無 `-classic` 後綴的 skill）。
- **與 prc-bureaucratese 明確區分**：
  - **PRC**：白話 + 政治意識形態 + 排比口號 + 數字化套語
  - **ROC classic**：文言 + 敬語層級 + 單字省略 + 程序套語
  - 共享「中文公文」大類但內涵本質不同。
