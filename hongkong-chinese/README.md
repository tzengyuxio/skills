# Hong Kong Chinese — 港式中文

將文字轉換為港式中文（港式書面語）——以現代漢語為骨架，融合粵語底層、英語接觸與香港特有社區詞，涵蓋港式詞彙（巴士/的士/雪櫃/唇膏）、港式音譯（荷里活/梳化/三文治）、傳承詞（食/飲/睇/瞓）、選擇性粵字（啲/嘢/搞掂）、頻繁中英夾雜（開 meeting / send email）、詞序倒置（質素/私隱/擠擁）。

## 功能

依照 checklist 將文字轉為港式中文風格，涵蓋以下模式：

| 類別 | 項目 |
|------|------|
| **語法模式** (G1-G5) | 西化結構、有/冇+動詞、畀/俾、動詞+得/唔、句末粵式助詞 |
| **詞彙模式** (V1-V9) | 港式常用詞、港式音譯、傳承詞、選擇性粵字、中英夾雜、社區詞、詞序倒置、同形異義、感情色彩差異 |
| **結構模式** (S1-S4) | 量詞差異、公文套語、英文名稱保留、粵式感嘆 |

支援三段強度：

| 強度 | 說明 |
|------|------|
| **輕** | 港式常用詞 + 港式音譯 + 少量中英夾雜（1-2 個英文詞/段）。維持標準中文骨架。像港商務 email。 |
| **中**（預設） | 全套 + 傳承詞（食/飲/睇/瞓）+ 選擇性粵字（啲/嘢/點解/搞掂）+ 頻繁中英夾雜 + 詞序倒置 + 社區詞。像 Apple Daily / Ming Pao 專欄。 |
| **重** | 中度 + 粵式助詞（㗎/啦/囉）+ 唔/到 可能補語 + 密集中英夾雜 + 放開使用粵字。仍保留「是/不/的/了/他」骨架，可區別於粵語書面化。 |

## 範疇界線（重要）

本 skill **不是粵語書面化**。關鍵差異：

| | 港式中文（本 skill） | 粵語書面化（不在本 skill 範圍） |
|---|---|---|
| 是 / 不 / 沒 | 保留 | 全改「係 / 唔 / 冇」 |
| 的 / 了 | 保留 | 全改「嘅 / 咗」 |
| 他 / 這 / 那 | 保留 | 全改「佢 / 呢 / 嗰」 |
| 食 / 飲 / 睇 / 啲 / 嘢 | 可散用 | 全用 |

也就是說，港式中文的讀者不分南北都能辨認為「帶港味的書面中文」；完整粵文則需要粵語知識才能讀通。

## 使用方式

```
/hongkong-chinese path/to/article.md
```

**觸發詞：** `/hongkong-chinese`、「寫成港式中文」、「港式中文化」、「翻成港式中文」、「港式書面語」、「港味」

支援任何語言輸入，轉為港式中文輸出。

## 運作流程

1. **載入** — 讀取檔案或行內文字
2. **轉換** — 依 checklist 逐項套用港式中文模式
3. **輸出** — 另存為 `_hongkong` 後綴新檔，附 3-5 組前後對比、港式詞彙 glossary

## 範例

| 原文（標準書面中文） | 港式中文（Medium） |
|---|---|
| 我坐公車去冰箱專賣店面試。 | 我搭巴士去雪櫃專門店見工。 |
| 我們去餐廳吃飯喝茶。 | 我哋去餐廳食飯飲茶。 |
| 他在辦公室開會。 | 佢喺 office 開 meeting。 |
| 請盡快回覆這封郵件。 | 請 ASAP reply 呢封 email。 |
| 他昨天被老闆炒了。 | 佢琴日畀老闆炒咗魷魚。 |
| 這個提案的素質很高。 | 呢個 proposal 嘅質素幾高。 |

## 設計原則

- **書面語域，非口語方言**：港式中文是 HK 書面中文的地域變體，根據邵敬敏等語言學家界定為「現代漢語的地域、社會、功能變體」
- **保留中文骨架**：「是/不/的/了/他」不換成粵字，以區別於完整粵語書面化
- **中英夾雜不隨機**：辦公室詞彙、科技動作詞、評價詞、專有名詞是 code-mixing 熱區
- **港式音譯固定**：荷里活不寫好萊塢、梳化不寫沙發、三文治不寫三明治
- **不做 caricature**：以 Apple Daily、Ming Pao、HK01、LIHKG 的真實書寫為範本，不做誇張「港腔」戲仿
