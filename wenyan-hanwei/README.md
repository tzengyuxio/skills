# Wenyan-Hanwei — 改寫為漢魏史傳體

將現代中文改寫為漢魏史傳體的歷史敘事散文。

## 功能

模擬司馬遷《史記》、班固《漢書》、陳壽《三國志》的敘事風格，追求「以事見人、寓褒貶於敘事」的質感。20 項 checklist 涵蓋：

- **語法 (G1-G6)** — 傳記起首「X者Y人也」、紀事時序、為…所…被動、賓語前置、簡筆白描、論贊體
- **詞彙 (V1-V6)** — 精確動詞（趨/詣/遣/拜）、稱謂代詞、春秋筆法、虛詞、否定詞、語氣詞
- **風格 (S1-S5)** — 以事見人、以言見性、敘事簡潔、古典紀時、對比映襯
- **時代標記 (R1-R3)** — 消除把/被、尊卑稱謂（崩/薨/卒/死）、去現代連詞

## 使用方式

```
/wenyan-hanwei path/to/article.md
```

**觸發詞：** `/wenyan-hanwei`、「史傳體」、「史記風格」、「漢書體」、「三國志體」

## 運作流程

1. **載入** — 讀取檔案或行內文字
2. **評估** — 識別傳主、關鍵事蹟、適合的形式（本紀/列傳/載記）
3. **改寫** — 依 checklist 逐項轉換，遵循 6 條改寫原則
4. **審查** — 二輪自檢：重掃 checklist + 通讀語感
5. **輸出** — 另存為 `_hanwei` 後綴新檔
