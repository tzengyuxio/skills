# Wenyan-Hanwei — 示範

展示 `wenyan-hanwei` skill 將現代中文改寫為漢魏史傳體的效果。源文來自 [`examples/source-texts/`](../../examples/source-texts/)。

本 skill 為單一風格輸出（無強度分層），仿《史記》《漢書》《三國志》的紀傳語感。

---

## 範例：差不多先生傳（短摘錄）

源文：[mr-chabuduo.md](../../examples/source-texts/mr-chabuduo.md)（胡適 1924，公有領域）

### 原文

> 你知道中國最有名的人是誰？提起此人，人人皆曉，處處聞名。他姓差，名不多，是各省各縣各村人氏。你一定見過他，一定聽過別人談起他。差不多先生的名字天天掛在大家的口頭，因為他是中國全國人的代表。
>
> 差不多先生的相貌和你和我都差不多。他有一雙眼睛，但看的不很清楚；他有兩隻耳朵，但聽的不很分明；有鼻子和嘴，但他對於氣味和口味都不很講究。他的腦子也不小，但他的記性卻不很精明，他的思想也不很細密。
>
> 他常常說：「凡事只要差不多，就好了。何必太精明呢？」
>
> 他小的時候，他媽叫他去買紅糖，他買了白糖回來。他媽罵他，他搖搖頭說：「紅糖白糖，不是差不多嗎？」
>
> 他在學堂的時候，先生問他：「直隸省的西邊是哪一省？」他說是陝西。先生說：「錯了。是山西，不是陝西。」他說：「陝西同山西，不是差不多嗎？」

### 改寫（漢魏史傳體）

> 差不多者，姓差，名不多，不知何郡人也。其族甚廣，遍於天下，各省各邑皆有之。凡中國之人，莫不識其名，莫不聞其事，蓋舉國之所代表者也。
>
> 其貌與常人無異。目能視而不明，耳能聽而不聰，有鼻有口，而於氣味飲食皆不甚辨。腦雖不小，記性鈍而思慮疏。
>
> 常語人曰：「凡事差不多即可矣，何必太精明哉？」
>
> 少時，母使買紅糖，乃持白糖歸。母責之，搖首曰：「紅糖白糖，非差不多乎？」
>
> 及就學，師問曰：「直隸之西何省？」對曰：「陝西。」師曰：「誤矣，是山西，非陝西也。」不多曰：「陝西與山西，非差不多乎？」

**字數比**：原文 ~330 字 → 改寫 ~210 字（≈64%）

## 改寫重點

| 規則 | 出處 | 改寫 |
|---|---|---|
| G1 傳記起首 | 「你知道中國最有名的人是誰？」 | 「差不多者，姓差，名不多，不知何郡人也」 |
| G2 紀事時序 | 「他小的時候」 | 「少時」 |
| G5 簡筆白描 | 「他有一雙眼睛，但看的不很清楚」 | 「目能視而不明」 |
| V1 精確動詞 | 「叫他去買」 | 「使買」 |
| V2 稱謂代詞 | 「他」 | 名字「不多」直呼 |
| V5 否定詞 | 「不是」 | 「非」 |
| V6 語氣詞 | 「呢？」 | 「哉？」「乎？」 |

## 風格特色

- **傳記起首**：「X 者，姓 X，名 X，X 郡人也」固定句式
- **客觀記事**：去除「你」「我」直接呼語，第三人稱敘述
- **精煉動詞**：說→曰、叫→使、罵→責
- **時序連詞**：少時、及…、嘗、是時
- **論贊體（可選）**：篇末加「太史公曰」「贊曰」
