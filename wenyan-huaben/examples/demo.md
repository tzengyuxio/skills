# Wenyan-Huaben — 示範

展示 `wenyan-huaben` skill 將現代中文改寫為話本小說體的效果。源文來自 [`examples/source-texts/`](../../examples/source-texts/)。

本 skill 為單一風格輸出（無強度分層），仿《水滸傳》《三國演義》《紅樓夢》《警世通言》的章回體說書語感。

> **特別注意**：話本是**敘事型擴寫**，不像先秦諸子體那樣壓縮——說書人插話、場景鋪墊、人物動作描寫會讓字數**膨脹至原文 130–180%**。

## 範例：差不多先生傳（短摘錄）

源文：[mr-chabuduo.md](../../examples/source-texts/mr-chabuduo.md)（胡適 1924，公有領域）

### 原文

> 你知道中國最有名的人是誰？提起此人，人人皆曉，處處聞名。他姓差，名不多，是各省各縣各村人氏。你一定見過他，一定聽過別人談起他。差不多先生的名字天天掛在大家的口頭，因為他是中國全國人的代表。
>
> 差不多先生的相貌和你和我都差不多。他有一雙眼睛，但看的不很清楚；他有兩隻耳朵，但聽的不很分明；有鼻子和嘴，但他對於氣味和口味都不很講究。他的腦子也不小，但他的記性卻不很精明，他的思想也不很細密。
>
> 他常常說：「凡事只要差不多，就好了。何必太精明呢？」
>
> 他小的時候，他媽叫他去買紅糖，他買了白糖回來。他媽罵他，他搖搖頭說：「紅糖白糖，不是差不多嗎？」
>
> 他在學堂的時候，先生問他：「直隸省的西邊是哪一省？」他說是陝西。先生說：「錯了。是山西，不是陝西。」他說：「陝西同山西，不是差不多嗎？」

### 改寫（話本小說體）

> **第一回　差不多敷衍半生事　圓通師糊塗一命亡**
>
> 詩曰：
>
> 　世人皆道差不多，萬事糊塗任蹉跎。
>
> 　到頭精細方知貴，悔入黃泉恨奈何。
>
> 話說這中華大地，幅員遼闊，人物眾多，奇人異士不可勝數。然天下之人，論起名聲最大、傳播最廣的，卻非甚麼帝王將相、文人才子，而是一位姓差、名不多的先生。列位看官，你道此人是哪一省哪一縣人氏？說來也怪，各省各縣各村，處處都有他的蹤跡，人人都識得他的名姓。你若問起旁人來，不拘男女老少，提起「差不多先生」五個字，無人不知，無人不曉。只因他便是這中國全國上下的代表人物，這也是後話了。
>
> 看官聽說，這差不多先生的相貌，與你我並無甚麼兩樣。他也有一雙眼睛，只是看東西不甚清楚；也有兩隻耳朵，只是聽聲音不甚分明；鼻子嘴巴俱全，卻對那氣味口味全然不去講究。他的腦袋倒也不小，只是那記性不甚精明，思量事情也不甚細密。
>
> 這差不多先生平日最愛說的一句話，便是：「凡事只要差不多，便好了。何必太認真呢？」看官且記住這句話，後面自有用處。
>
> 卻說差不多先生年幼之時，有一日，他母親差他出門去買紅糖。那差不多先生跑將出去，在街上買了一包白糖，高高興興提將回來。他母親接過看時，心下惱怒，罵道：「叫你去買紅糖，怎的買了白糖回來？」差不多先生卻搖了搖頭，笑嘻嘻說道：「媽，紅糖同白糖，不是差不多嗎？」他母親聽了，被噎得說不出話來。
>
> 且說差不多先生後來入了學堂唸書。一日，先生考問功課，問道：「直隸省的西邊是哪一省？」差不多先生想也不想，張口便答道：「是陝西。」先生聽了，搖頭說道：「錯了。是山西，不是陝西。」差不多先生歪著頭道：「陝西同山西，不是差不多嗎？」先生氣得鬍子直翹，竟拿他無法。

**字數比**：原文 ~330 字 → 改寫 ~660 字（≈200%，符合話本擴寫慣例）

## 改寫重點

| 規則 | 出處 | 改寫 |
|---|---|---|
| S1 回目對偶 | （無） | 「差不多敷衍半生事　圓通師糊塗一命亡」 |
| S2 開場詩 | （無） | 「世人皆道差不多…」7 言絕句 |
| G1 說書人起頭 | 「你知道…」 | 「話說這中華大地…列位看官，你道…」 |
| G2 場景轉換 | 段落分隔 | 「卻說」「且說」「話分兩頭」 |
| G4 對話標記 | 「他媽罵他」 | 「他母親接過看時，心下惱怒，罵道：…」 |
| V1 人物稱謂 | 「他」 | 「這差不多先生」「那差不多先生」 |
| V3 動作動詞 | 「跑去」「拿來」 | 「跑將出去」「提將回來」 |
| V5 描寫套語 | 「他媽很生氣」 | 「他母親聽了，被噎得說不出話來」 |

## 風格特色

- **回目對偶**：每回標題以對偶句點題，如「X 事　Y 人」格式
- **入話詩詞**：開頭以詩、詞或楹聯起興
- **說書人介入**：「列位看官」「看官聽說」「卻說」「且說」「話分兩頭」
- **文白夾雜**：核心敘述半文言（「跑將」「歪著頭」），對話保留口語
- **動詞補語**：「跑將出去」「提將回來」「說將起來」（V「將」+ 趨向）
- **設套式套語**：「但見…」「只見…」「正是那…時分」
- **回末懸念**：每回末尾「畢竟…如何，且聽下回分解」「正是：[詩句]」
