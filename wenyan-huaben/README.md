# Wenyan-Huaben — 改寫為話本小說體

將現代中文改寫為宋元明清話本小說 / 章回小說的敘事風格。

## 功能

模擬《水滸傳》《三國演義》《紅樓夢》《西遊記》的說書人敘事體，追求「說書人開口、看官側耳」的臨場質感。核心特徵是**文白夾雜**——敘述偏半文言，對話偏白話口語。

20 項 checklist 涵蓋：

- **語法 (G1-G6)** — 說書人起頭（話說/卻說）、場景轉換、文白夾雜、對話標記、懸念公式、把被兼用
- **詞彙 (V1-V6)** — 人物稱謂（那廝/官人/娘子）、半文言詞彙、動作動詞、量詞時間、程式化描寫、否定祈使
- **風格 (S1-S5)** — 回目對聯、有詩為證、說書人插話、敘事者議論、只見/但見場景描寫
- **時代標記 (R1-R3)** — 去現代連接詞、時代稱謂、避免穿越詞彙

改寫目標字數為原文的 **120-150%**——話本比古文鋪陳，但不灌水。

## 使用方式

```
/wenyan-huaben path/to/article.md
```

**觸發詞：** `/wenyan-huaben`、「話本體」、「說書體」、「水滸傳風格」、「三國演義體」、「章回小說」

## 運作流程

1. **載入** — 讀取檔案或行內文字
2. **評估** — 識別敘事情境、角色、場景轉換機會
3. **改寫** — 依 checklist 逐項轉換，遵循 6 條改寫原則
4. **審查** — 二輪自檢：重掃 checklist + 說書語感確認 + 字數比檢查
5. **輸出** — 另存為 `_huaben` 後綴新檔
